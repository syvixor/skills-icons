# Skills Icons에 기여하기

Skills Icons에 기여해 주시려는 관심에 감사드립니다! 여러분의 시간과 노력에 진심으로 감사드리며, 여러분의 기여는 이 프로젝트를 모든 사용자에게 더 나은 방향으로 발전시키는 데 큰 도움이 됩니다.

기여해 주시는 내용이 프로젝트의 목표와 기준에 부합하도록, 아래의 가이드를 꼭 읽어주세요.

## 기여하는 방법

### 1. 버그 신고하기

버그나 문제가 발견되면 다음 단계를 따라주세요:

- 먼저 동일한 내용의 이슈가 이미 등록되어 있는지 확인하세요.
- 아직 보고되지 않은 버그라면, 새 이슈를 생성해 등록해주세요.

### 2. 아이콘 기여하기

#### 아이콘 템플릿

- [일반 아이콘](../../.github/ICON_TEMPLATES/icon.svg)
- [테마 아이콘](../../.github/ICON_TEMPLATES/themed_icon.svg)

새 아이콘을 기여하고 싶다면, 다음 단계를 따라주세요:

- 제공된 아이콘 템플릿을 다운로드해 활용하세요. 이 템플릿들은 프로젝트 전체의 아이콘 스타일을 통일하기 위해 마련되었습니다.

- 아이콘이 프로젝트의 기존 디자인 원칙을 준수하도록 해주세요.

- **크기**, **스타일**, **선명도**에 특히 신경 써 주세요.

- 아이콘 파일은 /icons 디렉터리에 저장해야 합니다. 또한 아이콘 파일 이름은 네이밍 규칙에 맞게 지정해주세요 (예: iconname.svg).

### 3. Pull Request 검토 과정

Pull Request를 제출하면, 메인테이너가 이를 검토하며 추가 설명이나 수정 사항을 요청할 수 있습니다. 피드백이나 변경 요청에 적극적으로 응답해 주세요.

## 행동 강령

이 프로젝트에 기여함으로써, 여러분은 존중과 포용을 기반으로 한 소통 기준을 명시한 [행동 강령](../../.github/CODE_OF_CONDUCT.md)을 따르는 데 동의하는 것으로 간주됩니다.

## 라이선스

이 프로젝트에 기여하는 경우, 여러분의 기여 내용은 [프로젝트 라이선스](../../LICENSE) 하에 배포되는 데 동의하는 것으로 간주됩니다.
