# Συνεισφορά στο Skills Icons

Ευχαριστούμε για το ενδιαφέρον σας να συνεισφέρετε στο Skills Icons! Εκτιμούμε τον χρόνο και την προσπάθειά σας. Με τη συνεισφορά σας, βοηθάτε στη βελτίωση αυτού του έργου για όλους.

Παρακαλούμε διαβάστε τις παρακάτω οδηγίες για να διασφαλίσετε ότι οι συνεισφορές σας ευθυγραμμίζονται με τους στόχους και τα πρότυπα του έργου.

## Πώς να Συνεισφέρετε

### 1. Αναφορά Σφαλμάτων

Εάν βρείτε κάποιο σφάλμα ή πρόβλημα, παρακαλούμε ακολουθήστε τα εξής βήματα:

- Αναζητήστε εάν υπάρχει ήδη κάποιο issue για να δείτε αν το σφάλμα έχει ήδη αναφερθεί.
- Εάν το σφάλμα δεν έχει αναφερθεί ακόμα, ανοίξτε ένα νέο issue

### 2. Συνεισφορά Εικονιδίων

#### Πρότυπα Εικονιδίων

- [Κανονικό Εικονίδιο](../../.github/ICON_TEMPLATES/icon.svg)
- [Θεματικό Εικονίδιο](../../.github/ICON_TEMPLATES/themed_icon.svg)

Εάν θέλετε να συνεισφέρετε ένα νέο εικονίδιο, παρακαλούμε ακολουθήστε τα εξής βήματα:

- Κατεβάστε και χρησιμοποιήστε τα παρεχόμενα πρότυπα εικονιδίων. Αυτά τα πρότυπα έχουν σχεδιαστεί για να διασφαλίζουν ομοιομορφία στο στυλ των εικονιδίων σε όλο το έργο.

- Βεβαιωθείτε ότι το εικονίδιό σας τηρεί τις αρχικές σχεδιαστικές αρχές του έργου.

- Δώστε προσοχή στο **μέγεθος**, το **στυλ** και την **ευκρίνεια**.

- Τα εικονίδια πρέπει να τοποθετούνται στον κατάλογο /icons. Βεβαιωθείτε ότι το αρχείο του εικονιδίου ονομάζεται σύμφωνα με τη σύμβαση ονοματοδοσίας (π.χ. iconname.svg).

### 3. Διαδικασία Ελέγχου Pull Request

Μόλις υποβάλετε ένα pull request, οι διαχειριστές θα το εξετάσουν και ενδέχεται να ζητήσουν διευκρινίσεις ή αλλαγές. Παρακαλούμε να ανταποκρίνεστε σε τυχόν σχόλια ή αιτήματα για αλλαγές.

## Κώδικας Δεοντολογίας

Συνεισφέροντας σε αυτό το έργο, συμφωνείτε να ακολουθείτε τον [Κώδικα Δεοντολογίας](CODE_OF_CONDUCT.md), ο οποίος θέτει τις προσδοκίες για επικοινωνία με σεβασμό και συμπερίληψη.

## Αδειοδότηση

Συνεισφέροντας σε αυτό το έργο, συμφωνείτε ότι οι συνεισφορές σας θα αδειοδοτηθούν υπό την [άδεια του έργου](../../LICENSE).
