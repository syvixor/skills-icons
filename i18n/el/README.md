| Παράμετρος | Περιγραφή                                       | Υποχρεωτικό | Προεπιλογή |
| ---------- | ----------------------------------------------- | ----------- | ---------- |

### Σκοτεινή & Φωτεινή Λειτουργία 🌗

Το `Skills Icons` υποστηρίζει πλέον αυτόματη ανίχνευση θέματος — τα εικονίδια θα προσαρμόζονται απρόσκοπτα στη σκοτεινή 🌙 ή φωτεινή ☀️ λειτουργία του συστήματός σας χωρίς παραμέτρους ή χειροκίνητη ρύθμιση.

Αυτή η συμπεριφορά τροφοδοτείται από το ενσωματωμένο media query `prefers-color-scheme` του `CSS`, το οποίο ανιχνεύει την τρέχουσα προτίμηση θέματος του χρήστη και προσαρμόζει τα χρώματα SVG ανάλογα.

### Χρήση Docker 🐳

Αυτή η ενότητα παρέχει οδηγίες για τη δημιουργία και εκτέλεση του έργου `Skills Icons` χρησιμοποιώντας Docker. Ακολουθήστε τα παρακάτω βήματα για να τοποθετήσετε σε κοντέινερ και να διαχειριστείτε την εφαρμογή αποτελεσματικά.

#### Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε εγκαταστήσει τα εξής:

- Docker (έκδοση 18.09 ή νεότερη)

#### Δημιουργία της Εικόνας Docker

Για να δημιουργήσετε την εικόνα Docker για το `Skills Icons`, ακολουθήστε αυτά τα βήματα:

1. Ανοίξτε ένα τερματικό και πλοηγηθείτε στον κατάλογο.
2. Εκτελέστε την ακόλουθη εντολή για να δημιουργήσετε την εικόνα:

```bash
docker build -t skills-icons .
# ή
sudo docker build -t skills-icons .
```

#### Εκτέλεση του Κοντέινερ Docker

Μόλις δημιουργηθεί η εικόνα, μπορείτε να την εκτελέσετε σε ένα κοντέινερ:

1. Εκτελέστε την ακόλουθη εντολή:

```bash
docker run -p 3000:3000 skills-icons
# ή
sudo docker run -p 3000:3000 skills-icons
```

Αυτή η εντολή αντιστοιχίζει τη θύρα 3000 του κεντρικού υπολογιστή σας στη θύρα 3000 του κοντέινερ, επιτρέποντάς σας να έχετε πρόσβαση στο `Skills Icons` στο http://localhost:3000.
